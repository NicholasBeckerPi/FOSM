# FOSM
FOSM (stand for File Organizer and Send Mail) is a automation script , it organizes your files by date and send a ziped file to your email account.

## REQUIREMENTS

Python 3.7 or + installed

## RUNNING

```bash
python3.7 FOSM.py
```
